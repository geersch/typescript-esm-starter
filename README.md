## Description

Get started quickly with this ESM-enabled TypeScript starter repository. Configured with Vitest for testing, along with ESLint for code linting and prettier for formatting.

## Installation

```bash
$ yarn
```

### Running the app

```bash
# development
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Linting

```bash
$ yarn lint
```

## Formatting code

```bash
$ yarn format
```
